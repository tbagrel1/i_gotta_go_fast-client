## Afficher du code

Pour afficher du code sur une ligne, on peut utiliser les caractères `\`code\``
Pour afficher du code sur plusieurs lignes, on peut faire :  
```python
def est_pair(nombre):
    if nombre % 2 == 0:
        return True
    else:
        return False
```
